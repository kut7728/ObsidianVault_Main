### Single Pass

: 밑의 것들은 한번만 계산된다

  

### Constraints Go Down

: 제약은 부모로 부터 자식으로 전가된다

- Constraint는 크기의 제약으로 4가지가 있다
    1. 최대 높이
    2. 최소 높이
    3. 최대 너비
    4. 최소 너비

  

### Sizes Go Up

: 크기는 자식으로부터 부모에게 전달된다 (크기 정보 전달)

  

### Parent Sets Position

: 부모가 자식의 위치를 정해준다

  

> [!important] 자식 위젯이 어디 정렬 되는지 모를때는 자식 위젯의 크기가 무시될 수 있다 → 그냥 부모랑 같은 크기로 출력됨
