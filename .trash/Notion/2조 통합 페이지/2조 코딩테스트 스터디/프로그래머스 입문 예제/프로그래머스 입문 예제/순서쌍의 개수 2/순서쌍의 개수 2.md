### 나의 해답

```Swift
func solution(_ n:Int) -> Int {
    var answer = 0
    for i in 1...Int(sqrt(Float(n))) {
        if n % i == 0 {
            answer += (i*i == n) ? 1 : 2
        }
    }
    return answer
}
```

이전에 비슷한걸 해봐서.. 최소공배수? 그거였나

아무튼 전체를 도는것 보다 N의 제곱까지만 돌아도 모든 답을 찾을 수 있으니 반복을 덜할수 있다.

마지막에 삼항연산자는 만약 4는 (2*2) 쌍이 만들어질 수 있는데, 이때 2는 하나만 더해져야하기때문에 처리해줬음
