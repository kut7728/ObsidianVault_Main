# 회귀 모델

- 지도학습, 연속된 값을 예측
- 주어진 데이터셋에 최대한 가까운 직선을 그리는 것

  

# 분류 모델

- 지도학습, 불연속적인 값을 예측
- 데이터에 정답인 Label이 주어짐
- 주어진 데이터셋을 둘로 잘 나누는 직선 찾는 것

- 분류 모델을 만드는 알고리즘 중 하나

- 질문과 분기점 정하기를 데이터를 이용해서 학습함
